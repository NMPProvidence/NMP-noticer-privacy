Privacy Policy for NMP Notes (NMP Notatnik)

Effective Date: January 15, 2026

Application: NMP Notes / NMP Notatnik (note-taking app)

1. Data Collection and Storage

NMP Notes does not collect, store, transmit, or share any personal data or usage data. The application works entirely offline.

All notes created in the app are stored in a private SQLite database on the user's device that is not accessible to other applications. The developer does not have access to note content and does not process it in any way. The application does not communicate with any external servers operated by the developer.

The app does not use:

  • advertising or ad networks
  • analytics or tracking tools
  • developer-managed cloud services or data synchronization
  • user accounts or registration
  • subscriptions or in-app payments

2. Password Protection

Users may protect selected notes with a password. Passwords are stored locally on the user's device and are used solely to secure access to notes within the application.

Passwords are never transmitted, shared, or sent outside the device. Important: If a password is lost, the protected note cannot be recovered, as the developer has no access to it.

3. Backups and Storage Access Framework (SAF)

The application allows users to manually create backups of their notes. NMP Notes utilizes the Android Storage Access Framework (SAF) to ensure maximum privacy and user control.

The application itself does not transmit any data outside the device. When creating a backup, the application passes the data to the Android system, and the user manually selects the final destination through the system file picker. Depending on the user's device configuration and preferences, this may include:

  • Internal device storage,
  • An external SD card,
  • Or any other location supported by the device's system file manager (e.g., third-party cloud providers like Google Drive, OneDrive, or Dropbox).

The developer has no access to the backup files, does not know their storage location, and does not receive any data from them. Backup passwords (if set by the user) are not stored or transmitted and cannot be recovered.

4. Data Deletion

All primary data is stored locally. Users can delete notes at any time from within the app. Uninstalling the application permanently removes all locally stored data. Manual backups created by the user in external locations (SD card, cloud) remain under the user's control and must be deleted manually.

5. Children's Privacy

NMP Notes is not intended for children under the age of 13. The app does not knowingly collect any data from children, as no data collection mechanisms are implemented.

6. Privacy Policy Changes

Any changes to this Privacy Policy will be published with future updates of the application.
